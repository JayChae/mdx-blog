# 간단한 블록체인 네트워크 구현

블록체인 네트워크는 노드들이 서로 연결되어 데이터를 공유하는 분산 시스템입니다.

## P2P 네트워크 기본 개념

블록체인 네트워크의 핵심은 노드 간의 직접적인 통신입니다. 각 노드는 네트워크의 일부로서:

- 트랜잭션을 생성하고 전파
- 블록을 검증하고 전파
- 네트워크의 합의에 참여

## WebSocket을 활용한 구현

```javascript
const WebSocket = require('ws')
const wss = new WebSocket.Server({ port: 8080 })

wss.on('connection', (ws) => {
  console.log('새로운 노드가 연결되었습니다')

  ws.on('message', (message) => {
    const data = JSON.parse(message)
    // 메시지 처리 로직
  })
})
```

## 네트워크 토폴로지

- **스타 토폴로지**: 중앙 허브를 통한 연결
- **메시 토폴로지**: 모든 노드가 서로 직접 연결
- **하이브리드**: 효율성과 안정성의 균형

## 보안 고려사항

네트워크 통신 시 고려해야 할 보안 요소들:

- 메시지 암호화
- 노드 인증
- DDoS 공격 방어
- 메시지 무결성 검증
