# 간단한 지갑 구현

지갑은 암호화폐를 저장하고 전송할 수 있는 소프트웨어입니다.

## 지갑의 역할

암호화폐 지갑은 다음 기능들을 제공합니다:

- **개인키/공개키 관리**: 암호화폐 소유권 증명
- **주소 생성**: 거래를 위한 고유 식별자
- **트랜잭션 생성**: 암호화폐 전송
- **잔액 조회**: 보유 암호화폐 현황

## 타원 곡선 암호화

Bitcoin과 같은 암호화폐에서 사용하는 ECC (Elliptic Curve Cryptography):

```javascript
const EC = require('elliptic').ec
const ec = new EC('secp256k1')

class Wallet {
  constructor() {
    this.keyPair = ec.genKeyPair()
    this.publicKey = this.keyPair.getPublic('hex')
    this.privateKey = this.keyPair.getPrivate('hex')
  }
}
```

## 주소 생성

공개키로부터 지갑 주소를 생성하는 과정:

1. **SHA-256 해싱**: 공개키를 SHA-256으로 해싱
2. **RIPEMD-160 해싱**: SHA-256 결과를 RIPEMD-160으로 해싱
3. **체크섬 추가**: 버전 바이트와 체크섬 추가
4. **Base58 인코딩**: 사람이 읽기 쉬운 형식으로 변환

## 개인키 관리

지갑의 가장 중요한 요소인 개인키 관리:

- **안전한 저장**: 암호화된 형태로 저장
- **백업**: 복구를 위한 시드 구문 사용
- **보안**: 하드웨어 지갑이나 멀티시그 사용
- **복구**: 시드 구문으로 지갑 복원

## 지갑 유형

### 1. 소프트웨어 지갑

- **데스크톱 지갑**: 컴퓨터에 설치
- **모바일 지갑**: 스마트폰 앱
- **웹 지갑**: 브라우저 기반

### 2. 하드웨어 지갑

- **콜드 스토리지**: 인터넷과 분리된 저장
- **보안 칩**: 개인키를 안전하게 보호

### 3. 종이 지갑

- **프라이빗 키 출력**: 종이에 개인키 인쇄
- **오프라인 저장**: 인터넷 연결 불필요
